# ImprovedISMap
Improved map system for HBS BattleTech.

# What it does

Utilizes CSV file (by default, modified from Sarna's latest SUCKit map generator data) to generate an expanded Inner Sphere map, overriding the StarSystemDef's coordinates and ownerID, based on in-game year.

Reuses the following existing assets from [InnerSphereMap_data](https://github.com/Morphyum/InnerSphereMap/tree/master/InnerSphereMap_data):
- StarSystem 3025
- Cast
- Emblems
- Factions
- Heraldry
- Logos
- Revisions to game assets
